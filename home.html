To show many of these skills in a comprehensive project, I created a full-stack application in JavaScript. The artifact demonstrates all three skills of software engineering and design, data structures and algorithms, and databases. Immediately below is the original code and resulting slideshow for the early product in the simulated development process for the client “SNHU Travel”. I continued the development process to complete the application using the MERN (MongoDB, Express, React, and Node JS) stack. I first started with requirements that were not yet created but were priorities for the simulated client. These were communicated in user stories and expectations. Based on this information, I created the web application layout and features as if I were working with stakeholders/clients. For software engineering and design, I developed an interactive frontend component and a supporting backend server API. The React components enabled user account functionality, navigating/ searching vacations, and updating information as needed. This is supported by many data structure and algorithm considerations in the Express/Node JS server improving app performance with using arrays to filter and limit data and through paginating search results. The MongoDB database for the app contains user data and vacation data. Users create an account with specific preferences, and these are used when creating their accounts to reference the databases. Users have many operations available to them including creating an account, modifying their account preferences, researching price inputs, and updating their top vacation choices. Finally, the app is secured both on the frontend and backend. User inputs are limited to protect against malicious entries, data integrity is checked on the server side to ensure it is correct/valid, schema creation to restrict information to specific formats/ types, user’s passwords are encrypted and never stored in API responses, and error handling is fully controlled. Many of app’s features are demonstrated in the “Enhancement” section below.
